The code is arranged in subdirectories for each recipe in the chapter. Each recipe chapter contains the JDeveloper
application workspaces as directories. In each application woskspace directory there are the relevant workspace projects. 

To open a specific workspace from JDeveloper, select Open Application... from the Application Navigator browse
through the available workspaces and open the relevant application.

Reusable ADF Library JARs are saved in a directory called "ReUsableJARs".
